JavaScript Array Tutorial

In this tutorial, you will learn how to work with JavaScript arrays.

Creating Arrays

To create a new array, use the following syntax:

var myArray = [1, 2, 3, 4, 5];

Accessing Array Elements

To access an element in an array, use the index number:

var firstElement = myArray[0]; // returns 1

Note that array indices start at 0.

Array Example

Here is an example of how to use an array:

// create a new array
var fruits = ['apple', 'banana', 'orange'];

// print the first element
console.log(fruits[0]); // outputs 'apple'

// add a new element to the end of the array
fruits.push('kiwi');

// remove the last element from the array
fruits.pop();

// loop through the array and print each element
for (var i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}
